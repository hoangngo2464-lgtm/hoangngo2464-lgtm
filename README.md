# Xin chào, mình là Ngô Hoàng

Mình đang học và phát triển các dự án về **IoT, hệ thống nhúng, Arduino/ESP32, điều khiển robot và ứng dụng phần mềm hỗ trợ kỹ thuật**. GitHub này là nơi mình lưu lại các bài thực hành, đồ án và sản phẩm mình tự xây dựng trong quá trình học.

## Mình đang tập trung vào

- Lập trình Arduino, ESP32 và các hệ thống nhúng
- Xe dò line, né vật cản, điều khiển động cơ và cảm biến
- Ứng dụng thuật toán PID, Kalman Filter trong robot
- RTOS và lập trình đa nhiệm cho vi điều khiển
- Python GUI và các thuật toán xử lý dữ liệu
- Kết nối IoT với Firebase, Telegram và ứng dụng mobile/web

## Dự án nổi bật

### Xe dò line và né vật cản

Repo: [dolineweb](https://github.com/hoangngo2464-lgtm/dolineweb)

Dự án robot sử dụng Arduino/C++ để dò line, né vật cản và điều khiển động cơ. Project có nhiều phiên bản thử nghiệm, bao gồm bản dùng Kalman Filter để lọc dữ liệu và cải thiện độ ổn định khi robot di chuyển.

Điểm nổi bật:

- Điều khiển robot dò line
- Né vật cản bằng cảm biến
- Áp dụng PID/Kalman trong xử lý tín hiệu
- Có tài liệu báo cáo, hình ảnh và sơ đồ dây

### Tìm hiểu RTOS

Repo: [ROTS](https://github.com/hoangngo2464-lgtm/ROTS)

Dự án tìm hiểu hệ điều hành thời gian thực RTOS và làm bài tập áp dụng trên Arduino/Proteus. Repo phù hợp để lưu lại quá trình học về task, delay, lập lịch và xử lý nhiều tác vụ trên vi điều khiển.

Điểm nổi bật:

- Thực hành lập trình đa nhiệm
- Mô phỏng Proteus
- Code Arduino/C++
- Tài liệu báo cáo đi kèm

### Mã hóa LZW bằng Python

Repo: [mahoaLZW](https://github.com/hoangngo2464-lgtm/mahoaLZW)

Ứng dụng Python mô phỏng thuật toán nén dữ liệu LZW, có giao diện Tkinter để mã hóa, giải mã và hiển thị kết quả.

Điểm nổi bật:

- Cài đặt thuật toán LZW
- Giao diện GUI bằng Tkinter
- Có minh họa kết quả mã hóa và giải mã
- Phù hợp cho môn kỹ thuật đa phương tiện/xử lý dữ liệu

### Firmware IoT BECA

Repo: [beca](https://github.com/hoangngo2464-lgtm/beca)

Repo lưu firmware cho dự án IoT BECA. Mục tiêu tiếp theo của repo này là bổ sung source code, hướng dẫn build và tài liệu cấu hình để người xem có thể hiểu rõ cách hoạt động của hệ thống.
